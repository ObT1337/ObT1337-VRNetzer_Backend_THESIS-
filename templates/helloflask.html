Hello flask - super basic
Open the folder VRNetzerWebApp in visual studio code
In VSCode open a terminal and run .\buildandRUN.ps1 (for windows, mac_buildandrun.sh for mac) in the VRNetzerWebApp directory. That should create a venv and install all requirements. If all went well the console will show something like "server is running"

Add a route
add the following code to app.py under ### HTML ROUTES ### Line 40
app.py
@app.route("/mynewroute")
def mynewroute(): 
    return "hello"
after saving this the console should print that it restarted. That indicates that hotreloading works, meaning the server reflects the changes you make as soon as you safe a file

click me http://127.0.0.1:5000/mynewroute
We created a http route that returns the string hello.

Now lets create a new html file in /templates
mynewroute.html
<!DOCTYPE html>
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
</head>
<body>
 <div>
  <h1>Helloflask</h1>
 </div>
</body>
</html>
and return that instead.
app.py
@app.route("/mynewroute")
def mynewroute(): 
    return render_template("mynewroute.html")
http://127.0.0.1:5000/mynewroute
We created a http route that returns the html template we just made.
